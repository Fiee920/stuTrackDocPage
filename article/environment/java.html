Java17环境配置 ​

基于CentOS7做命令演示，根据实际使用的Linux发行版灵活调整。

1.下载Java ​

https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html

下载对应的Java版本，例如阿里云的X86版CentOS7需要安装Linux x64 Compressed Archive

2.使用SFTP工具上传到Linux服务器 ​

3.上传到成后解压 ​

shell
tar -zxvf jdk-17.0.7_linux-x64_bin.tar.gz

4.（可选）移动到合适位置 ​

shell
mv jdk-17.0.7 /usr/local/

5.编辑环境变量配置 ​

shell
vim /etc/profile

追加以下内容：

shell
# 根据实际Java版本号选择
export JAVA_HOME=/usr/local/java/jdk-17.0.10
export CLASSPATH=.:$JAVA_HOME/lib
export PATH=$PATH:$JAVA_HOME/bin

按下ESC，输入:wq!保存并退出

6.刷新配置文件 ​

shell
source  /etc/profile
