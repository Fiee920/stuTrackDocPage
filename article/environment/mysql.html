MySQL 8配置（基于Docker） ​

基于CentOS7做命令演示，根据实际使用的Linux发行版灵活调整。

由于配置MySQL过于繁琐，所以依赖于Docker容器部署MySQL

1.Docker仓库搜索MySQL ​

shell
docker search mysql

2.Docker仓库拉取MySQL ​

shell
docker pull mysql:8.0.36
# 或者使用docker pull mysql 拉取最新版本

3.查看本地仓库镜像是否下载成功 ​

shell
docker images mysql:8.0.36

4.安装运行MySQL容器 ​

shell
docker run -p 3306:3306 --name mysql8.0.36 -e MYSQL_ROOT_PASSWORD=root -d mysql:8.0.36 --lower_case_table_names=1
# 第一个3306是本地主机的端口映射到docker容器端口
# 这里就设置好了MySQL的root用户的密码
# --lower_case_table_names=1放在最后，表示配置大小写不敏感

5.查看MySQL容器运行情况 ​

shell
docker ps

6.Docker登录MySQL ​

shell
docker exec -it mysql8.0.36 bash  # 这里的名字也可以写id的前三位
mysql -uroot -p
